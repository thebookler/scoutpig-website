Privacy Policy

Last Updated: February 2025

1. Introduction

Welcome to ScoutPig! Your privacy is important to us. This Privacy Policy explains how we collect, use, and protect your data when you use our app and website, including integration with Amazon’s Selling Partner API (SP-API).

By using ScoutPig, you agree to the collection and use of your data as described in this policy. If you have any questions, please contact us at support@scoutpig.com.

2. Information We Collect

2.1 Personal Information

  • Email Address (for login and marketing communications)
  • Name (for account identification)
  • Amazon Seller ID & MWS/SP-API Tokens (for accessing Amazon’s Selling Partner API)
  • Location (for service personalization)
  • Payment Information (if applicable)

2.2 Automatically Collected Data

  • Product Lookups & Search Data: We store product searches, ASIN/ISBN lookups, and pricing history for analytics.
  • Usage Data: We track how you use the app for performance optimization.
  • Device & Log Data: Includes IP address, device type, and browser version for security.

2.3 Third-Party Data Collection

  • Google Analytics & Firebase Analytics: For tracking user engagement.
  • Firebase Authentication & Cloud Storage: For secure authentication and data storage.
  • Amazon SP-API: To retrieve real-time pricing and product analytics.

3. How We Use Your Data

We use your data to:

  • Provide and improve the ScoutPig app.
  • Fetch and display Amazon product pricing and analytics.
  • Securely store and manage your Amazon API tokens.
  • Personalize product recommendations.
  • Send marketing emails (opt-out available).
  • Maintain security and prevent fraud.

4. Amazon Selling Partner API Data Handling

4.1 Data We Access from Amazon SP-API

With your explicit authorization, we access:

  • Product Listings & Pricing Data (ASIN lookup, lowest price, FBA & MFN offers).
  • Seller Account Data (Amazon Seller ID, Marketplace ID).

4.2 How We Secure Amazon API Tokens

We securely store SP-API access and refresh tokens using encrypted storage. Tokens are only used for:

  • Authenticated API calls on behalf of the seller.
  • Retrieving real-time pricing and product data.
  • Ensuring seamless login without requiring repeated authentication.

We do not store or process any Amazon customer data.

5. Data Sharing & Third-Party Services

We do not sell your data. We only share necessary data with:

  • Google & Firebase Services (for authentication and analytics).
  • Amazon SP-API (for product analytics and pricing data).
  • Payment Processors (if applicable).

6. How Users Can Control Their Data

Account Deletion: If you wish to delete your account, email support@scoutpig.com.

Amazon API Token Removal: You can revoke access anytime via Amazon Developer Central.

Email Preferences: Unsubscribe from marketing emails anytime.

7. Data Security Measures

We use strong security practices, including:

  • Encryption: Amazon API tokens and sensitive data are stored securely.
  • Authentication: We use Firebase Authentication for secure access.
  • Access Controls: Only authorized personnel handle API data.

8. Children's Privacy

ScoutPig is designed for Amazon sellers. We do not knowingly collect data from children under 13 years old.

9. Updates to This Privacy Policy

We may update this policy periodically. Any major updates will be communicated via email or in-app notifications.
